# نحوه استفاده :

برای استفاده دو کد زیر را در تگ body کد های html خود قرار دهید :

```html
<input id="url" value="https://github.com/aminrngbr1122">
<input id="s" value="5">
```

value s مقدارش ثانیه ریدایرکته @
value url لینک وب سایتی که میخواید بهش ریدایرکت بشه @
